Messaging

The web app starts with a sign up screen followed by a login page. When the user is logged in then they can select into messaging and use this feature. First a screen will display 2 options of selecting contacts or messaging screen. Contacts will allow the user to search a user and send and message him directly. Message screen will display all the latest messages received by the user. There will be an option to send a voice message. The chat window will display all the previous messages with timestamps. All messages are stored in the server.

Texting
